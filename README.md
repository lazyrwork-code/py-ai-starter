# AI Learning Path: Python for AI

Project ini dirancang sebagai panduan belajar Python untuk pengembangan AI, mulai dari dasar hingga tingkat lanjut (RAG & Agentic Workflow).

## Roadmap Belajar

### 1. [Basics](./01-basics/)
- Sintaks dasar Python
- Data structures (List, Dict, Set)
- List Comprehension & Functions
- Pengenalan Virtual Environment

### 2. [FastAPI](./02-fastapi/)
- Dasar-dasar Web API
- Pydantic untuk validasi data
- Integrasi backend untuk aplikasi AI

### 3. [AI Integration](./03-ai-integration/)
- Menggunakan SDK OpenAI/Claude
- Prompt Engineering dasar
- Manajemen Environment Variables (.env)

### 4. [LangChain & RAG](./04-langchain/)
- Pengenalan LangChain Framework
- Retrieval Augmented Generation (RAG)
- Vector Databases
- Simple AI Agent

## Cara Memulai
1. Clone repository ini.
2. Buat virtual environment: `python -m venv venv`.
3. Aktifkan venv:
   - Windows: `.\venv\Scripts\activate`
   - Mac/Linux: `source venv/bin/activate`
4. Install dependensi: `pip install -r requirements.txt`
